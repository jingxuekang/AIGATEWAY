# 菜单权限说明

## 管理员菜单（Admin）

### 首页
- Dashboard - 查看整体数据统计

### 访问与鉴权
- **渠道管理** - 管理上游 AI 服务商渠道（OpenAI、Claude 等）
- **API Keys** - 管理所有用户的 API Keys（查看、创建、删除、启用/禁用）
- **申请审批** - 审批用户的 API Key 申请（通过/拒绝）

### 用户与计费
- **用户管理** - 管理所有用户账号、配额
- **充值记录** - 查看所有用户的充值记录

### 模型管理
- **模型管理/定价** - 配置可用模型、设置价格
- **模型发布&订阅** - 管理模型的发布和订阅关系

### 运维 & 观测
- **调用日志** - 查看所有 API 调用记录
- **任务日志** - 查看系统后台任务执行情况

### 工具
- **系统设置** - 系统全局配置
- **接口文档** - API 接口文档
- **Playground** - 在线测试 AI 模型

---

## 普通用户菜单（User）

### 首页
- Dashboard - 查看个人使用统计

### 我的密钥
- **我的申请** - 申请新的 API Key、查看申请状态

### 模型订阅
- **可用模型** - 查看和订阅可用的 AI 模型

### 使用记录
- **调用日志** - 查看自己的 API 调用记录
- **充值记录** - 查看自己的充值记录

### 工具
- **接口文档** - API 接口文档
- **Playground** - 在线测试 AI 模型

## 实现说明

### 前端权限控制
- 通过 `localStorage.getItem('role')` 获取当前用户角色
- 根据角色显示不同的菜单项
- 位置：`frontend/src/components/Layout/index.tsx`

### 后端权限控制（待实现）
- 需要在后端添加权限拦截器
- 验证用户角色和访问权限
- 普通用户只能访问自己的数据

### 切换角色测试
在浏览器控制台执行：
```javascript
// 切换为管理员
localStorage.setItem('role', 'admin')
location.reload()

// 切换为普通用户
localStorage.setItem('role', 'user')
location.reload()
```

## 页面行为差异

### 申请审批 / 我的申请页面

**管理员视图（申请审批）：**
- 页面标题：申请审批
- 显示所有用户的申请
- 表格列：ID、申请人、Key名称、申请模型、申请理由、状态、审批意见、申请时间、操作
- 操作按钮：通过、拒绝（仅待审批状态显示）
- 无"新建申请"按钮

**普通用户视图（我的申请）：**
- 页面标题：我的申请
- 只显示自己的申请
- 表格列：ID、Key名称、申请模型、申请理由、状态、申请时间
- 无操作按钮（不能审批）
- 有"新建申请"按钮

### 其他需要区分的页面

**Dashboard：**
- 管理员：显示全局统计（所有用户、所有调用）
- 用户：显示个人统计（自己的配额、调用量）

**调用日志：**
- 管理员：可以查看所有用户的调用日志
- 用户：只能查看自己的调用日志

**充值记录：**
- 管理员：可以查看所有用户的充值记录
- 用户：只能查看自己的充值记录

**模型发布&订阅 / 可用模型：**
- 管理员：管理模型发布、查看订阅情况
- 用户：查看可用模型、订阅模型
